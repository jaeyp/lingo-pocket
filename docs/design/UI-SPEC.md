### ✅ 1. 리스트뷰 레이아웃 ✅ **확정**

**레이아웃:**
```
┌─────────────────────────────────────┐
│  English Surf [+]                   │ ← 타이틀, 추가
│  (⇄)                           [▶] │ ← 토글(아이콘), 낱장보기
├─────────────────────────────────────┤
│  (Chip) [Sort: Random ▼]  (Chip) [Diff: All ▼]  │ ← Filter Chips (Horizontal Scroll)
├─────────────────────────────────────┤
│  ┌───────────────────────────────┐ │║
│  │ I'm looking forward to seeing │ │║
│  │ you.                          │ │║ ← 스크롤바
│  └───────────────────────────────┘ │║
│  ┌───────────────────────────────┐ │║
│  │ There's a lot of stuff you    │ │║
│  │ guys do that could ruin your  │ │║
│  │ trip.                         │ │║
│  └───────────────────────────────┘ │║
│  ┌───────────────────────────────┐ │║
│  │ I was willing to talk to      │ │║
│  │ anyone.                       │ │║
│  └───────────────────────────────┘ │║
│  ...                               │║
└─────────────────────────────────────┘
```

**확정 사항:**
- ✅ **세로 리스트** (한 줄에 카드 1개)
- ✅ **Endless List** (무한 스크롤)
- ✅ **스크롤바** 표시
- ✅ **카드 높이:** 문장 길이에 따라 가변 (전체 문장 표시)
- ✅ **텍스트 표시:** 스타일 없는 **Plain Text**로 표시 (가독성 최우선)
- ✅ **Learning Mode:**
  - `EN → KO` (영어 학습) ← **기본값**
  - `KO → EN` (영어 작문)
- ✅ **Sort Options:**
  - `order` (사용자 지정 순서)
  - `difficulty` (난이도 순서)
  - `random` (랜덤) ← **기본값**
- ✅ **Difficulty Filter:**
  - `all` (전체)
  - `beginner` (초급)
  - `intermediate` (중급)
  - `advanced` (고급)
- ✅ **제스처 (Swipe-to-Action):**
  - **Left Swipe:** 삭제 (Delete) - 빨간색 배경, 쓰레기통 아이콘 (Confirm Dialog 필요)
  - **Right Swipe:** 편집 (Edit) - 파란색 배경, 연필 아이콘

### ✅ 3. ID vs Order 필드 ✅ **확정**

**확정 사항:**
- ✅ **별도 order 필드 추가**
- ✅ `id`: 영구적인 고유 식별자 (변경 안 함)
- ✅ `order`: 표시 순서 (변경 가능)

**JSON 구조:**
```json
{
  "id": 5,
  "order": 1,
  "original": {
    "text": "I'm looking forward to seeing you.",
    "styles": [
      {"type": "bold", "start": 4, "end": 19},
      {"type": "highlight", "start": 23, "end": 29}
    ]
  },
  "translation": "당신을 만나기를 기대하고 있어요.",
  "difficulty": "beginner",
  "examples": [],
  "notes": ""
}
```

**Freezed 모델 업데이트 필요:**
```dart
@freezed
class Sentence with _$Sentence {
  const factory Sentence({
    required int id,
    required int order,
    required SentenceText original, // ← SentenceText 객체
    required String translation,
    required Difficulty difficulty, // ← Enum
    @Default([]) List<String> examples,
    @Default('') String notes,
  }) = _Sentence;
}
```

**장점:**
- ✅ ID는 영구적 → 학습 진도 추적 가능
- ✅ order만 변경 → 안전하고 명확

### ✅ 4. 카드 뷰 버튼 배치 ✅ **확정**

**확정 사항:**
- ✅ 버튼을 **AppBar로 이동**

- ✅ **우측:** [×] (닫기)
- ✅ **카드 전체 터치 → 앞/뒤 뒤집기**

## 📝 추가 기능

### ✅ 학습 모드 (역방향 학습) ✅ **확정**

**개요:**
- 원문만 보고 학습 ↔ 번역 보고 원문 추론
- 두 가지 모드 전환 가능

**확정 사항:**

**1. 모드 옵션**
- ✅ `Orig → Trans` (원문 학습) - 기본값
- ✅ `Trans → Orig` (작문/추론)
- ❌ `Random` 제외

**2. 모드 선택 UI**
- ✅ **토글 버튼** (리스트 화면 상단)
  - **아이콘 (⇄)** 클릭 시 즉시 전환
  - 드롭다운 메뉴 없음, 한 번 클릭으로 전환
- ✅ **마지막 모드 기억** (SharedPreferences)
  - 앱 종료 후 재시작해도 유지

**3. 다국어 지원**
- ✅ **언어 중립적 설계**
- ✅ JSON: `translation` (String) - `translations` (Map) 아님
- ⏳ 나중에 필요하면 다국어 확장

**4. 리스트뷰 표시**
- ✅ 모드에 따라 앞면 텍스트만 표시
  - `Orig→Trans`: 원문 (Original)
  - `Trans→Orig`: 번역 (Translation)

**UI 디자인:**

**리스트 화면:**
```
┌─────────────────────────────────────┐
│  English Surf [+]                   │ ← 타이틀, 추가
│  (⇄)                           [▶] │ ← 토글 버튼(아이콘), 낱장보기
├─────────────────────────────────────┤
│  (Chip) [Sort: Random ▼]  (Chip) [Diff: All ▼]  │
├─────────────────────────────────────┤
│  ┌───────────────────────────────┐ │
│  │ I'm looking forward to...     │ │ ← Orig→Trans 모드
│  └───────────────────────────────┘ │
│  ┌───────────────────────────────┐ │
│  │ 당신을 만나기를 기대하고...   │ │ ← Trans→Orig 모드
│  └───────────────────────────────┘ │
└─────────────────────────────────────┘

[ (⇄) ] 클릭 시:
→ 모드 즉시 전환
```

**카드 뷰 - Orig→Trans 모드:**
```
앞면: I'm looking forward to seeing you.
뒷면: 당신을 만나기를 기대하고 있어요. + 상세정보
```

**카드 뷰 - Trans→Orig 모드:**
```
앞면: 당신을 만나기를 기대하고 있어요.
뒷면: I'm looking forward to seeing you. + 상세정보
```

### 기타 조정/개선사항

- ✅ **Dark Mode 지원:**
  - 시스템 설정에 따라 자동 전환
